# work-weekly-brief

A classic Vibe Work skill: turn a dataset into a one-page executive brief.

1. Read the dataset the user points at; compute the 3 most important facts
   (extremes, trend, notable outlier).
2. **Create a Canvas** containing:
   - a title and a one-sentence takeaway,
   - a "Highlights" section with the 3 facts,
   - a ranked table of the key entities,
   - one chart that carries the main message.
3. Keep it to one page. End with a suggested next action.

*(Note: in Vibe Code, the canvas-to-cli skill converts every Canvas step
above into terminal artifacts and renderings - same content, same order.)*

Vibe Pulse6
Terminal-native MLOps

W&B needs a server and a browser. We don't.

  • Run tracking · every run logged to runs/index.jsonl
  • Run comparison · vibe-plot --compare overlays any metric across experiments
  • Agent-driven sweeps · “try 3 learning rates and compare” · no sweep server, the agent is the orchestrator
  • ASCII PlantUML diagrams · architecture & dependency graphs without leaving the terminal
  • vibe-arena · LMArena-style model comparison in the console: Mistral vs OpenAI, latency & tok/s charted
  • Any data · CSV, JSON streams, Excel, SQLite (--sql) · and Canvas-style tables (--table)
